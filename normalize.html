Normalize.css + Better Typography

Normalize.css itself is opinionated and this modified version is even more opinionated, but in a very good way. This code is based on the v8.0.0 of Normalize.css, with additions of basic typographic styles. Its purpose is to serve as the standard reset for starting any minmal design projects, all styles aim to mimic the browser defaults with a touch of polish.

Spacing Rules

Elements are spaced using single-direction margin declarations, in this case, margin-bottom. Base spacing unit is equal to the leading (line-height) minus the space between mean line and the top bearing line: 1.65 - (1.65 - 1) / 2. That means block elements are spaced 1.325rem in-between vertically.

Headings and lists would be exempt from the standard spacing because they should vary to indicate clear information grouping.
